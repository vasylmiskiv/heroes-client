# Heroes app client

- Download or clone the repository

## Installation
### Install the dependencies and devDependencies and start the server.
- Open the folder:
```sh
cd heroes-api
```
- Open your favorite Terminal and run these commands:
```sh
npm i
```
If you have yarn:
```sh
yarn
```

## Launch the app

- Open your favorite Terminal and run these commands:
```sh
npm run start
```
If you have yarn:
```sh
yarn start
```
- Open in the browser http://localhost:5173
